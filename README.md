# Starter theme for Hexo static site generator

## Project structure

```text
.
├── _config.yml // Configuration file
├── languages   // Stores language variables
├── layout      // Stores layouts of the theme
├── scripts     // Stores scripts that will be imported directly into pages during build process
├── source      // Assets and files that will be built and published to final output
    ├── assets    // Static assets that can be just dumped into public folder
        ├── icons
        └── images
    ├── js   // JavaScript files that will be build into scripts folder.
    └── scss // SCSS files that will be build into css files
```
